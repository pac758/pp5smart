ระบบบริหารจัดการโรงเรียน (School MIS)

สารบัญ

1. ภาพรวมและบทบาทผู้ใช้ 2. การเข้าสู่ระบบ 3. ข้อมูลนักเรียน 4. เวลาเรียน (เช็คชื่อ) 5. บันทึกคะแนน 6. ประเมินผล 7. เอกสาร / รายงาน (ปพ.5, ปพ.6) 8. ระบบผู้ปกครอง 9. ตั้งค่าระบบ (Admin) 10. คำถามที่พบบ่อย

1. ภาพรวมและบทบาทผู้ใช้

ระบบบริหารจัดการโรงเรียน (School MIS) เป็น Web App ที่ทำงานบน Google Apps Script ร่วมกับ Google Sheets ครอบคลุมงานบริหารจัดการข้อมูลนักเรียน เวลาเรียน คะแนน ประเมินผล และพิมพ์เอกสารรายงาน

บทบาทผู้ใช้

บทบาทสิทธิ์การใช้งาน
Adminเข้าถึงทุกเมนู รวมถึงตั้งค่าระบบ จัดการผู้ใช้ จัดการวันหยุด คลังคะแนน ส่งออกระบบ
Teacherเช็คชื่อ บันทึกคะแนน ประเมินผล พิมพ์รายงาน จัดการข้อมูลนักเรียน
Parentดูผลการเรียนและข้อมูลบุตรหลานผ่านระบบผู้ปกครอง

2. การเข้าสู่ระบบ

  1. เปิดลิงก์ Web App ที่ได้รับจากผู้ดูแลระบบ
  2. กรอก ชื่อผู้ใช้ และ รหัสผ่าน
  3. กดปุ่ม เข้าสู่ระบบ
  4. ระบบจะพาไปหน้า Dashboard โดยอัตโนมัติ
หมายเหตุ: หากลืมรหัสผ่าน ให้ติดต่อผู้ดูแลระบบ (Admin) เพื่อรีเซ็ตรหัสผ่าน

เปลี่ยนรหัสผ่าน

ไปที่เมนู บัญชีผู้ใช้ > เปลี่ยนรหัสผ่าน → กรอกรหัสผ่านเก่าและใหม่ → กดบันทึก

3. ข้อมูลนักเรียน

รายชื่อนักเรียน

เลือก ชั้น และ ห้อง เพื่อดูรายชื่อ แสดงข้อมูลรหัส คำนำหน้า ชื่อ-นามสกุล สามารถค้นหาและกรองได้

เพิ่มนักเรียน Admin

กรอกข้อมูล: รหัสนักเรียน, คำนำหน้า, ชื่อ, นามสกุล, ชั้น, ห้อง แล้วกดบันทึก

จัดการข้อมูลนักเรียน Admin Teacher

แก้ไขข้อมูลรายคน เปลี่ยนสถานะ (ปกติ/จำหน่าย/ย้ายออก/พ้นสภาพ) หรือลบข้อมูล

นำเข้ารายชื่อ DMC Admin Teacher

นำเข้าข้อมูลนักเรียนจากไฟล์ DMC (ระบบจัดเก็บข้อมูลนักเรียนรายบุคคล) รองรับรูปแบบมาตรฐาน

ส่งออกรายชื่อ PDF Teacher

เลือกชั้น/ห้อง แล้วส่งออกเป็น PDF สำหรับพิมพ์

4. เวลาเรียน (เช็คชื่อ)

เช็คชื่อรายวัน Teacher

  1. เลือก ชั้น และ ห้อง
  2. เลือก วันที่ ที่ต้องการเช็คชื่อ
  3. เลือกสถานะ: มา / ลา / ขาด สำหรับนักเรียนแต่ละคน
  4. กดบันทึก

เช็คชื่อรายเดือน Teacher

แสดงตารางเช็คชื่อทั้งเดือน (แนวนอน: วันที่, แนวตั้ง: รายชื่อ) คลิกที่ช่องเพื่อเปลี่ยนสถานะ แสดงจำนวนวันที่เช็คชื่อจริงใต้หัวข้อ "มา"

พิมพ์เวลาเรียนรายเดือน Teacher

ส่งออกตารางเวลาเรียนรายเดือนเป็น PDF

สรุปมาเรียนรายภาคเรียน Teacher

แสดงสรุปสถิติมา/ลา/ขาด ของนักเรียนทั้งภาคเรียน

จัดการวันหยุด Admin

เพิ่ม/แก้ไข/ลบวันหยุดราชการและวันหยุดพิเศษ

5. บันทึกคะแนน

สร้างตารางคะแนน Teacher

  1. เลือก ชั้น, ห้อง, วิชา
  2. ระบบสร้างชีตคะแนนพร้อมรายชื่อนักเรียนอัตโนมัติ
  3. แต่ละภาคเรียนมี: คะแนนครั้งที่ 1-4 (ระหว่างเรียน), ครั้งที่ 5 (กลางภาค), ซ่อม, ครั้งที่ 6-9, ครั้งที่ 10 (ปลายภาค)

บันทึกคะแนนรายวิชา Teacher

เลือกชีตคะแนน → กรอกคะแนนนักเรียนแต่ละคน → ระบบคำนวณรวม+เกรดอัตโนมัติ → กดบันทึก

รายงานคะแนนรายวิชา Teacher

แสดงสรุปคะแนนพร้อมค่าสถิติ (เฉลี่ย สูงสุด ต่ำสุด)

สรุปผลการเรียนรายชั้น Teacher

แสดงสรุปผลการเรียนทุกวิชาของนักเรียนในชั้นที่เลือก

คลังคะแนนรวม Admin

ดูและจัดการข้อมูลคะแนนทั้งหมดในระบบ

6. ประเมินผล

ประเมินอ่าน คิด เขียน

ประเมินความสามารถด้านการอ่าน คิดวิเคราะห์ เขียน — ระดับ: ดีเยี่ยม / ดี / ผ่าน / ไม่ผ่าน

ประเมินคุณลักษณะอันพึงประสงค์

ประเมิน 8 ประการตามหลักสูตร เช่น รักชาติ มีวินัย ใฝ่เรียนรู้ ฯลฯ

ประเมินกิจกรรมพัฒนาผู้เรียน

ประเมินลูกเสือ-เนตรนารี ชุมนุม จิตอาสา — ระดับ: ผ่าน / ไม่ผ่าน

ประเมินสมรรถนะสำคัญ

ประเมิน 5 ด้าน: สื่อสาร คิด แก้ปัญหา ใช้เทคโนโลยี ฯลฯ

บันทึกความคิดเห็นครู

เขียนข้อสังเกต/ความคิดเห็นเกี่ยวกับนักเรียน ข้อมูลนี้จะแสดงใน ปพ.6

ขั้นตอน: เลือกชั้น/ห้อง → เลือกระดับสำหรับนักเรียนแต่ละคน → กดบันทึก

7. เอกสาร / รายงาน

รวมส่งออกรายงาน

หน้ารวมลิงก์สำหรับส่งออกรายงานทุกประเภท

ปก ปพ.5

สร้างปกเอกสาร ปพ.5 (แบบบันทึกผลการเรียน) เป็น PDF แสดงข้อมูลโรงเรียน ชั้น ห้อง ครูประจำชั้น วิชา

ปพ.5 รวมเล่ม (PDF)

สร้าง ปพ.5 ฉบับเต็มรวมทุกวิชาของชั้นที่เลือก รวมคะแนน เกรด สถิติ ในเล่มเดียว

ปพ.6 - รายงานผลการพัฒนา

สมุดรายงานผลการพัฒนาคุณภาพผู้เรียน รวมผลการเรียน เวลาเรียน ประเมินคุณลักษณะ สมรรถนะ กิจกรรม และความคิดเห็นครู

รายงานหน้าเดียว

สรุปผลการเรียนรายบุคคลในหน้าเดียว เหมาะสำหรับแจกผู้ปกครองในวันประชุม

ก่อนพิมพ์รายงาน: ตรวจสอบว่ากรอกคะแนน, ประเมินผล, และบันทึกเวลาเรียนเรียบร้อยแล้ว

8. ระบบผู้ปกครอง

เฉพาะผู้ใช้ที่มีบทบาท Parent เท่านั้นที่จะเห็นเมนูนี้

  • ดูผลการเรียนของบุตรหลาน
  • ดูสถิติเวลาเรียน
  • ดูผลประเมินด้านต่าง ๆ
สำหรับ Admin: สร้างบัญชี Parent ที่เมนู จัดการผู้ใช้ โดยระบุบทบาทเป็น "parent" และผูกกับรหัสนักเรียนของบุตรหลาน

9. ตั้งค่าระบบ Admin

ตั้งค่าระบบ

แก้ไขข้อมูลพื้นฐาน: ชื่อโรงเรียน, ปีการศึกษา, ที่อยู่ ฯลฯ

จัดการโลโก้

อัปโหลดโลโก้โรงเรียนสำหรับใช้ในเอกสาร/รายงาน

จัดการรายวิชา

เพิ่ม/แก้ไข/ลบรายวิชา กำหนดชื่อวิชา รหัสวิชา จำนวนหน่วยกิต

จัดการครูประจำชั้น

กำหนดครูประจำชั้นแต่ละห้อง ข้อมูลจะแสดงในรายงาน ปพ.5, ปพ.6

จัดการผู้ใช้

เพิ่ม/แก้ไข/ลบบัญชีผู้ใช้ กำหนดบทบาท (Admin/Teacher/Parent) รีเซ็ตรหัสผ่าน

เตรียมข้อมูลปีการศึกษาใหม่

สร้างชีตเช็คชื่อสำหรับทุกเดือนในปีการศึกษาใหม่ (พ.ค.-มี.ค.) สร้างเฉพาะชีตที่ยังไม่มี

เช็คอัปเดตระบบ

ตรวจสอบเวอร์ชันใหม่และแสดงวิธีอัปเดต

ส่งออกระบบให้โรงเรียนอื่น

เฉพาะโรงเรียนต้นแบบ — สร้าง Template สำหรับแจกจ่าย ระบบจะลบข้อมูลส่วนบุคคลออกก่อนส่ง

10. คำถามที่พบบ่อย

ระบบโหลดช้ามาก
Google Apps Script มี cold start ครั้งแรกอาจใช้เวลา 5-10 วินาที ครั้งต่อไปจะเร็วขึ้น หากช้าตลอดให้รีโหลดหน้าเว็บ หรือล้าง Cache
บันทึกคะแนนแล้วหาย
ตรวจสอบว่ากด "บันทึก" แล้วรอจนระบบแจ้ง "สำเร็จ" ก่อนปิดหน้า
เช็คชื่อแล้วไม่ขึ้นใน Dashboard
Dashboard มีระบบ cache ข้อมูลจะอัปเดตภายใน 5 นาที หรือรีโหลดหน้าใหม่
ลืมรหัสผ่าน
ติดต่อ Admin ให้รีเซ็ตที่เมนู "จัดการผู้ใช้"
อยากเปลี่ยน Dark Mode
คลิกไอคอนพระจันทร์/พระอาทิตย์ ที่มุมบนขวาของหน้าจอ
ปพ.5 ไม่แสดงข้อมูลคะแนน
ตรวจสอบว่าสร้างตารางคะแนนแล้ว กรอกคะแนนและกดบันทึกแล้ว เลือกชั้น/ห้อง/วิชาตรงกัน
จะอัปเดตระบบได้อย่างไร
ไปที่ ตั้งค่าระบบ > เช็คอัปเดตระบบ ระบบจะตรวจสอบและแสดงวิธีอัปเดต
